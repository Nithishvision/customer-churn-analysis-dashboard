# Customer Churn Analytics Dashboard

## Project Overview
This project analyzes customer churn behavior using Power BI. The dashboard provides insights into customer retention, churn patterns, internet service usage, payment methods, and contract types.

## Problem Statement
Customer churn is one of the major challenges faced by telecom companies. This dashboard helps identify the key factors influencing customer churn and provides business insights to improve customer retention.

## Dashboard Features
- Interactive Power BI dashboard
- Customer churn analysis
- Contract type analysis
- Internet service insights
- Payment method analysis
- KPI reporting
- Business recommendations

## Key Insights
- Customers with month-to-month contracts have higher churn rates
- Fiber optic users show maximum churn behavior
- Electronic check payment users are more likely to churn
- Customers without tech support and security services tend to leave more often

## Tools & Technologies
- Power BI
- DAX
- Excel
- Data Visualization
- Business Intelligence

## Business Recommendations
• Introduce attractive long-term contract plans to improve customer retention.

• Provide bundled service packages including Tech Support and Online Security to increase customer satisfaction.

• Develop targeted retention campaigns for high-risk customer segments, especially short-tenure customers.

• Improve customer engagement strategies for Fibre Optic users through personalized offers and better support services.

• Offer loyalty rewards and discounts to encourage long-term customer relationships.
